## Final Project: Complete Game Prototype

### Project Requirements

Create a playable game prototype demonstrating:

1. **Character System**
   - Player character with movement
   - Health, stamina, inventory
   - At least 2 abilities (GAS)

2. **Combat**
   - Melee weapon with combo
   - Ranged weapon/projectile
   - Damage feedback

3. **AI**
   - At least 2 enemy types
   - Patrol and combat behaviors
   - Death and respawn

4. **UI**
   - Main menu
   - In-game HUD
   - Pause menu
   - Inventory screen

5. **Game Loop**
   - Win/lose conditions
   - Score tracking
   - Save/load progress

6. **Polish**
   - Sound effects
   - Particle effects
   - Camera feedback

### Grading Criteria

| Category | Weight | Description |
|----------|--------|-------------|
| Functionality | 40% | All features working |
| Code Quality | 25% | Clean, organized, documented |
| Performance | 15% | 60+ FPS, no hitches |
| Polish | 20% | Effects, feedback, juice |
