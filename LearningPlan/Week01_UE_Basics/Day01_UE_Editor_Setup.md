## 核心概念（二八法则中的20%）

### 1. 项目结构
```
YourProject/
├── Source/           # C++源文件
├── Content/          # 资产（蓝图、材质等）
├── Config/           # 配置文件
├── Binaries/         # 编译后的二进制文件
└── Intermediate/     # 构建中间文件
```

### 2. 关键文件
- `YourProject.uproject` - 项目定义文件
- `YourProject.Build.cs` - 构建配置
- `YourProject.Target.cs` - 构建目标设置
